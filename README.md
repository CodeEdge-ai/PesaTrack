# PesaTrack 💚

> Personal Finance Tracker built for Kenya — KES/USD, M-Pesa SMS parsing, PAYE calculator, dark mode.

**Live:** https://codeedge-ai.github.io/PesaTrack/

## Features
- 💱 KES/USD toggle with live exchange rate
- 📱 M-Pesa SMS parser — paste SMS → auto-fill transaction
- 🧾 PAYE / NHIF / NSSF tax calculator (2024 Kenya rates)
- 📊 Analytics — Income vs Expenses, Savings Trend, Category Breakdown
- 🎯 Savings Goals with progress rings
- 💳 Debt Tracker with repayment logging
- 🔔 Bills & Reminders
- 🌙 Dark mode
- 📦 All data in localStorage — no backend needed

## Stack
React 18 · Recharts · Lucide · Vite

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Push to `main` → GitHub Actions auto-builds and deploys to GitHub Pages.
